========================
Insert Data into MongoDB
========================

.. default-domain:: mongodb

MongoDB provides the :method:`db.collection.insert()` method in the
:program:`mongo` shell to add documents to a collection. When
performing an insert operation, if the collection does not exist, the
insert operation will create the collection.
